## 🚀 快速开始

### 对于用户

1. 查看 说明文档.md 了解项目
2. 按照 新手入门指南.md 安装
3. 查看 安装配置.md 了解配置

### 对于开发者

1. 阅读 贡献指南.md 了解工作流程
2. Fork 项目到你的账号
3. 创建功能分支进行开发
4. 提交 Pull Request
